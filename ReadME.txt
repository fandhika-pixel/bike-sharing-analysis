# Proyek Analisis Data - Bike Sharing Dataset

## Deskripsi
Proyek ini merupakan hasil analisis data dari dataset Bike Sharing untuk memenuhi tugas akhir kelas Analisis Data Dicoding. Dashboard ini memberikan wawasan tentang performa penyewaan sepeda berdasarkan musim dan pengaruh faktor cuaca.

## Struktur Direktori
- **/dashboard**: Berisi dashboard.py dan data yang sudah dibersihkan (main_data.csv).
- **/data**: Berisi dataset asli (day.csv dan hour.csv).
- **notebook.ipynb**: File Jupyter Notebook untuk analisis data.
- **requirements.txt**: Daftar library yang dibutuhkan.
- **README.md**: Petunjuk penggunaan.

## Instalasi
1. Pastikan Anda memiliki Python terinstal di sistem Anda.
2. Instal library yang dibutuhkan dengan menjalankan perintah berikut di terminal/PowerShell:
   ```bash
   pip install -r requirements.txt


Cara Menjalankan Dashboard
Masuk ke direktori proyek melalui terminal atau PowerShell.

Jalankan aplikasi Streamlit dengan perintah berikut:

	streamlit run dashboard/dashboard.py
